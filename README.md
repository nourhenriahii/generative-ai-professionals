## 🎓 Key Learnings

1. **Prompt Engineering is Critical** — good prompts mean better AI outputs; iterative refinement and context matter greatly.
2. **AI Augments, Not Replaces** — AI works best with human judgment; human-in-the-loop is optimal.
3. **Automation Saves Real Time** — identified workflows can be automated with significant ROI.
4. **Data Quality Determines Results** — cleaning data is crucial; visualization reveals patterns.
5. **Continuous Learning is Necessary** — the AI field evolves rapidly, so staying updated is essential.
